# 猫咪私人助理 — 功能说明文档

## 一、项目概述

"猫咪私人助理"是一个基于 Web 的智能养猫管理工具。用户通过自然语言对话，记录和管理猫咪的日常事务，包括花费、健康事件、体重、库存、照片等。后端采用 FastAPI + SQLite，接入 DeepSeek 大模型的 Function Calling 能力，实现自然语言到数据库操作的自动转换。前端为单页应用（SPA），使用原生 JavaScript + Chart.js 实现交互式数据可视化和图片浏览。

- **当前版本端口**：8001
- **数据存储目录**：`D:\cat-assistant-data\`
- **照片存储目录**：`D:\cat-assistant-data\photos\`
- **PWA 支持**：manifest.json 配置，浏览器通知来源显示为"猫咪私人助理"

## 二、功能模块详解

### 2.1 对话交互（聊天记录）

- 用户通过聊天界面与猫咪助理沟通，助理自动理解意图并执行操作
- 支持快捷指令标签（打疫苗、买猫粮、记录体重、体重趋势、查看提醒）
- **对话历史持久化**：使用 localStorage 保存对话记录，刷新页面后自动恢复
- 支持清空对话记录功能
- 输入框自动伸缩（最大 120px 高度）
- 回车发送（Shift+Enter 换行）
- 加载动画（三点跳动）
- 照片上传后内联显示缩略图，点击可进入查看器
- **每次对话后自动刷新仪表盘和日历**（如果正在显示）
- **写入操作追踪**：/chat 响应返回 `data_changed` 字段，标记是否有数据被真实写入

### 2.2 仪表盘

点击顶部导航栏「📊」按钮打开。
- **猫咪档案卡片**：显示头像、名字、生日、品种、性别、状态标签。点击头像可上传/更换猫咪头像（覆盖 avatar.jpg，同时自动存入相册数据库）。点击简介可编辑，点击状态标签可更换状态
- **状态标签**：减肥中、生病中、恢复期、挑食中、活泼好动、安静嗜睡
- **即将到期事件**：未来 7 天内到期的疫苗/驱虫/就医提醒，点击可查看详情
- **库存不足**：低于阈值的物品数量统计
- **本月花费**：本月开销汇总（使用统一日期范围函数，与 Agent 查询一致），点击进入分类统计
- **总开销**：历史全部花费汇总
- **最近体重**：最新一条体重记录
- **体重变化趋势图**：内嵌折线图，自动渲染
- **分类趋势图**：下拉选择分类（猫粮/猫砂/零食/玩具/医疗/用品），点击查看该分类的月度花费折线图
- **5 秒自动轮询刷新**：仪表盘打开时每 5 秒自动拉取最新数据

### 2.3 日历视图

点击顶部导航栏「📅」按钮打开。
- **月视图日历**：展示当前月份的每一天，标记有事件的日期
- **事件标签**：疫苗💉、驱虫💊、就医🏥、拔牙/口腔🦷、生日🎂
- **点击日期**：查看当天所有事件的详细信息
- **拖拽调整**：支持将事件拖拽到另一个日期，自动更新 `next_due_date`
- **上月/下月切换**：箭头按钮浏览不同月份
- **今天按钮**：快速跳回当前月份
- **当天照片入口**：有照片的日期可点击查看当天上传的照片

### 2.4 浏览器通知

- **权限提示**：页面加载 3 秒后显示通知开启提示条
- **即将到期事件推送**：每天检查一次（页面加载 + 每 2 小时），有 7 天内到期事件时弹出浏览器通知
- **智能总结**：≤3 条事件逐条推送，>3 条汇总推送
- **防重复**：localStorage 记录已推送事件，同一天不重复推送
- **持久通知**：`requireInteraction: true`，通知不会自动消失
- **点击通知**：聚焦到应用窗口

### 2.5 花费管理

#### 2.5.1 记录花费
支持自然语言记录花费，如"今天买了渴望猫粮花了300块"。AI 自动识别类别、品牌、金额、日期。

**7 大花费类别**：猫粮、玩具、医疗、猫砂、零食、用品、其他

**去重机制**：日期+类别+品牌+金额+备注完全相同的记录自动跳过，防止 AI 多轮 Function Calling 重复提交。

#### 2.5.2 查询花费
- 按时间范围查询：本月、上月、近 3 个月、近半年、今年、全部
- 按类别筛选
- 自动对比上月花费（显示变化百分比）
- 品牌分析排行（品牌花费金额和次数 TOP10）

#### 2.5.3 修改花费（新增）
- 支持自然语言修改已有记录，如"把5月28号皇家猫粮的备注改成400g"
- AI 先调用 `query_expenses` 或 `search_expense` 找到记录 ID
- 再调用 `update_expense` 执行部分更新（只改用户提到的字段）
- 严禁 AI 假装修改——必须真正调用 update_expense 函数

#### 2.5.4 删除花费
- 支持自然语言删除（如"把刚才那笔猫粮删掉"）
- AI 先调用 `search_expense` 搜索匹配记录，展示给用户确认
- 用户确认后调用 `delete_expense` 执行删除
- 也提供 REST API 直接删除（`DELETE /api/expense/{id}` 和 `DELETE /api/expense/last`）

#### 2.5.5 统计数据一致性
- **统一日期范围函数** `get_month_range(period)`：所有 Agent 函数和 API 共用，确保月末边界一致（如"本月"始终 = 1号 ~ 月末最后一天）
- **统一统计函数** `get_expense_total()` 和 `get_expense_by_category()`：Agent 和仪表盘走同一段 SQL 逻辑，从根源杜绝数据不一致

#### 2.5.6 子分类系统

**医疗子分类**：
- 疫苗（关键字：疫苗、猫三联、妙三多）
- 驱虫（关键字：驱虫、大宠爱、海乐妙、爱沃克、博莱恩、福来恩、拜耳）
- 医保（关键字：医保、保险）
- 其他医疗

**用品子分类**：
- 猫砂盆、猫抓板、猫碗、猫包、牵引绳、猫梳子、伊丽莎白圈、猫砂铲、宠物垃圾袋、免洗手套湿巾、费洛蒙喷雾
- 其他用品

**零食/玩具/猫粮/猫砂**：按品牌名称自动分组

### 2.6 花费统计与可视化

#### 2.6.1 分类统计弹窗
- 展示全部时间各分类总花费 + 本月各分类花费
- 点击任一分类进入下钻详情，展示子分类/品牌分组
- 支持查看单个子分类/品牌的全部购买记录（日期、金额、备注）

#### 2.6.2 趋势图表（Chart.js 4.4.0）
- **单分类趋势**：折线图，展示某分类各月的花费变化
- **多分类对比**：所有分类在同一图表中对比，折线图
- **聊天内嵌图表**：当 AI 返回 `[CHART_DATA]` 标记时，聊天消息中自动渲染「查看趋势图」按钮，点击弹出图表弹窗
- **仪表盘图表**：下拉选择分类，实时渲染该分类趋势
- **分类详情图表**：在分类下钻弹窗底部内嵌小趋势图
- 图表颜色方案统一（猫粮橙色、猫砂紫色、零食粉色、玩具青色、医疗红色、用品蓝色、其他灰色）

### 2.7 事件记录与提醒

#### 2.7.1 事件记录
支持 5 种事件类型：
- **疫苗 (vaccine)**：默认下次到期日 +365 天
- **驱虫 (deworm)**：默认下次到期日 +90 天
- **就医 (vet_visit)**：无默认到期日
- **拔牙/口腔 (dental)**：无默认到期日
- **生日 (birthday)**：猫咪生日

#### 2.7.2 定时提醒
- 后台 asyncio 定时任务，每天 8:00 自动检查
- 启动时立即执行一次检查
- 提醒内容包括：
  - 未来 7 天内到期的事件（疫苗/驱虫/就医等），显示倒计时天数
  - 猫咪生日提醒（7 天内），随机 7 种趣味语气
  - 库存不足物品提示

#### 2.7.3 手动查询
- 用户可通过"有什么提醒吗"查询
- 可通过 API `/api/reminders` 手动获取

### 2.8 库存管理

- 5 个类别：猫粮、猫砂、药品、零食、用品
- 字段：物品名称、类别、当前数量、阈值、单位、备注
- **低于阈值自动提示**（更新库存时返回警告，定时提醒中包含）
- **猫粮消耗估算**（`calculate_food_days`）：
  - 幼猫（<1岁）按每天 60g/kg 估算
  - 成猫（1-10岁）按每天 50g/kg 估算
  - 老猫（>10岁）按每天 40g/kg 估算

### 2.9 体重记录

- 支持自然语言记录体重（如"猫咪体重5.2kg"）
- 查询体重变化趋势
- **智能分析**：
  - 变化 > 0.3kg → "体重呈上升趋势，猫咪可能偏胖了，建议控制饮食"
  - 变化 < -0.3kg → "体重呈下降趋势，需要关注猫咪健康"
  - 变化在 ±0.3kg 内 → "体重基本稳定，猫咪状态良好"
- 数据不足时提示需要更多记录

### 2.10 猫咪档案

- 信息：名字、生日、品种（奶牛猫）、性别（女）
- 生日自动计算年龄和距离下次生日的天数
- 头像上传（覆盖式保存为 `avatar.jpg`，自动同步到相册数据库）
- 支持通过对话更新猫咪信息
- **状态标签**：在仪表盘显示当前状态，可通过对话或 UI 更新
- **猫咪简介**：点击可在仪表盘直接编辑

### 2.11 相册管理

- 照片存储在 `D:\cat-assistant-data\photos\`
- 上传入口：聊天输入框旁的 📷 按钮
- **支持多张照片同时上传**：文件选择器添加 `multiple` 属性，Ctrl+点击多选，全部一次性提交。批量上传时文件名自动加序号避免冲突
- 照片元数据保存在 SQLite `photos` 表（文件名、原始名、上传日期、文件大小）
- 数据库记录与文件系统双重索引（补充数据库中缺失的文件系统照片）
- 支持格式：JPG、JPEG、PNG、GIF、WebP、BMP
- 头像文件（avatar.jpg）自动排除
- 聊天中上传的照片内联显示缩略图

### 2.12 图片查看器

- **日期分组浏览**：从相册网格点击，自动筛选同一天的所有照片
- **单张查看**：从聊天中的照片缩略图点击进入
- **滚轮切换**：鼠标滚轮上下滚动切换前后图片
- **箭头按钮**：左右箭头按钮切换图片
- **键盘导航**：← → 方向键切换，ESC 关闭
- **计数器显示**：当前张数/总张数，日期标签
- **操作提示**：多张图片时显示滚轮/箭头操作提示
- 深色背景（#1c1917）沉浸式浏览

### 2.13 玩具喜好

- 记录猫咪对玩具的喜好程度：喜欢 (like)、不喜欢 (dislike)、一般 (neutral)
- 自动记录最近使用日期
- 支持自然语言记录（如"它不喜欢逗猫棒"、"喜欢玩偶"）

### 2.14 出行模拟

- 用户说"我5月28-30日出差"时自动触发
- 检查项：
  - 猫粮库存是否足够旅行天数（按每天 50g 估算）
  - 猫砂库存是否充足（按每天 100g 估算）
  - 旅行期间是否有到期事件（疫苗/驱虫/就医）
- 给出补给建议和提前安排建议

### 2.15 驱虫计划动态调整

- 场景一：用户说"家里来了新猫" → 自动缩短驱虫间隔
- 场景二：用户说"带猫外出遛弯" → 自动缩短驱虫间隔
- 默认缩短 5 天，在原有 `next_due_date` 基础上提前
- 操作记录追加到事件备注中

## 三、技术架构

### 3.1 后端（Python / FastAPI）

| 组件 | 技术选型 |
|------|----------|
| Web 框架 | FastAPI |
| 数据库 | SQLite（`cat_assistant.db`，9 张表） |
| AI 接口 | DeepSeek API（`deepseek-chat` 模型） |
| 异步运行时 | asyncio + httpx |
| 定时任务 | asyncio Task（每天 8:00 循环） |
| 文件服务 | FastAPI StaticFiles |

#### 数据库表结构（9 张表）

| 表名 | 用途 |
|------|------|

| GET | `/api/upcoming_events` | 未来 7 天到期事件 |

### 3.2 前端（原生 HTML/CSS/JS）

| 组件 | 技术选型 |
|------|----------|
| 框架 | 无框架，原生 JavaScript |
| 图表 | Chart.js 4.4.0（CDN） |
| 数据持久化 | localStorage |
| 样式 | 纯 CSS（CSS 变量、动画、响应式） |

#### 关键设计
- **单页应用**：所有功能在一个 `index.html` 中实现，通过弹窗（Modal）切换
- **对话恢复**：页面加载时从 localStorage 恢复 `chatHistory`（API 上下文）和 `displayMessages`（渲染内容）
- **图表实例管理**：仪表盘和聊天弹窗各自独立维护 Chart.js 实例，切换时正确销毁旧实例
- **图片查看器**：维护 `allPhotosCache` 全局缓存，按日期分组，滚轮事件绑定在模态框上
- **仪表盘自动刷新**：打开时 5 秒轮询，对话后立即刷新
- **响应式**：适配移动端（最大宽度 600px 断点）
- **PWA**：manifest.json 配置，独立窗口显示，通知来源为"猫咪私人助理"

### 3.3 启动脚本（Windows）

`start.bat`：
- 设置 UTF-8 编码（`chcp 65001`）
- 修复 Anaconda sqlite3 DLL 路径
- 自动读取 `api_key.txt` 设置环境变量
- 精确杀死占用 8001 端口的进程（而非杀全部 Python）
- 启动 `python app.py`

## 四、使用方法

### 4.1 环境要求
- Python 3.8+
- 需要安装依赖：`pip install fastapi uvicorn httpx`
- DeepSeek API Key（在项目目录创建 `api_key.txt` 写入 Key，或设置环境变量 `DEEPSEEK_API_KEY`）

### 4.2 启动
1. 双击 `start.bat` 启动服务
2. 浏览器访问 `http://localhost:8001`
3. 手机访问：`http://<电脑IP>:8001`（需同一 WiFi）

### 4.3 基本对话示例

| 场景 | 对话示例 |
|------|----------|
| 记录疫苗 | "我家猫昨天打了猫三联疫苗" |
| 记录花费 | "今天买了渴望猫粮5.4kg，花了300块" |
| 修改记录 | "把5月28号皇家猫粮那条的备注改成400g" |
| 删除花费 | "把刚才那笔猫粮删掉" |
| 记录体重 | "猫咪体重5.2kg" |
| 查询花费 | "这个月猫粮花了多少钱？" |
| 查总花费 | "我一共花了多少钱？" |
| 查看趋势 | "猫粮每个月花多少？" 或 "看看趋势" |
| 查询提醒 | "最近有什么提醒吗？" |
| 出行模拟 | "我5月28号到30号出差" |
| 记录偏好 | "它不喜欢逗猫棒" |
| 更新库存 | "新买了一包10kg的猫砂" |
| 更新状态 | "顺顺在减肥" |
| 查看照片 | "展示5月25号的照片" |

### 4.4 界面操作

- **顶部导航栏**：
  - 🖼️ 相册：查看所有上传的猫咪照片
  - 📅 日历：月视图日历，查看和拖拽调整事件
  - 📊 仪表盘：查看猫咪综合数据
  - 🔄 清空对话：清除聊天记录
- **输入区域**：
  - 文本输入框（回车发送，Shift+Enter 换行）
  - 📷 上传照片按钮（支持 Ctrl+多选批量上传）
  - ➤ 发送按钮
- **快捷指令**：点击输入框上方的标签快速发送常见查询
- **仪表盘内操作**：
  - 点击头像区域上传/更换猫咪头像
  - 点击状态标签更换猫咪状态
  - 点击简介文字编辑猫咪简介
  - 点击「即将到期事件」查看详情
  - 点击「本月花费」或「总开销」查看分类统计
  - 分类统计中点击任一分类进入子分类详情
  - 子分类/品牌名称可继续点击查看全部购买记录
  - 下拉选择分类后点击「查看」渲染趋势图
- **日历操作**：
  - ← → 箭头切换月份，点击"今天"回到当前月
  - 点击有事件的日期查看当日详情
  - 拖拽事件卡片到另一个日期可调整到期日
- **相册操作**：
  - 点击任意照片进入查看器
  - 查看器中使用鼠标滚轮或左右箭头切换同一天的照片
  - 按 ESC 或点击 ✕ 关闭查看器
  - 键盘 ← → 方向键翻页
